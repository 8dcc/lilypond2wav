#+TITLE: lilypond2wav
#+AUTHOR: 8dcc

Converts a [[https://lilypond.org][LilyPond]] source file to a WAV audio file by synthesizing its notes.

* Features

- Parses a subset of LilyPond notation: single notes, chords, rests, dots,
  ties, relative mode, key signatures, and multiple languages.
- Two built-in synthesizers: a sine wave synthesizer and an additive harmonic
  synthesizer (default) with an ADSR envelope for a warmer tone.
- Tempo can be read from =\tempo= directives in the source file or overridden
  from the command line.
- Gate control (=-g=) shortens each note's sounding duration, producing a
  staccato-like effect.

* Dependencies

You will need =python-ly= and =numpy=:

#+begin_src bash
pip install python-ly numpy
#+end_src

* Usage

The basic usage is:

#+begin_src bash
python src/main.py INPUT.ly
#+end_src

The following options are supported:

| Short | Long       | Description                                          |
|-------+------------+------------------------------------------------------|
| =-h=  | =--help=   | Show the help message and exit.                      |
| =-o=  | =--output= | Output WAV path (default: input stem + =.wav=).      |
| =-s=  | =--synth=  | Synthesizer: =harmonic= (default) or =sine=.         |
| =-b=  | =--bpm=    | Tempo in BPM (overrides =\tempo= in the input file). |
| =-g=  | =--gate=   | Fraction of each note that sounds (default: 1.0).    |
